# Licensing Expo 2026 — 展商攻略

這是一個專為 Licensing Expo 2026 設計的單頁式網頁應用程式（Single Page Application），提供互動式的展商目錄，幫助使用者高效瀏覽、篩選並標記重點展商，最大化參展效益。

## ✨ 核心功能

*   **🌓 深淺色模式切換**：支援亮色（Light）與暗色（Dark）主題，適應不同環境與個人偏好。
*   **📊 數據總覽儀表板**：頂部提供 Tier 1 至 Tier 3 的展商數量統計，快速掌握重點目標。
*   **🔍 強大的搜尋與篩選系統**：
    *   **優先度篩選 (Tier)**：將展商分為 Tier 1（必訪）、Tier 2（優先）、Tier 3（探索）與 Tier 4（略過）。
    *   **類別篩選 (Category)**：支援依據 IP 屬性分類，如電玩 IP、動漫 IP、影視娛樂、運動品牌、生活風格、服務商等。
    *   **關鍵字搜尋**：可快速搜尋展商名稱或相關描述。
*   **🔀 彈性排序**：支援依據優先度（Tier）、名稱（A-Z）、類別（Category）及展位（Booth）進行排序。
*   **👀 雙重瀏覽模式**：
    *   **表格檢視 (Table View)**：適合快速掃視大量資訊。
    *   **卡片檢視 (Card View)**：提供更豐富的視覺呈現與詳細說明。
*   **⭐ 書籤功能**：點擊星號可將展商加入書籤，並透過「書籤」按鈕快速過濾出已標記的目標。
*   **📱 響應式設計 (RWD)**：完美適配桌機、平板與手機等各式裝置，並在行動版提供專屬的側邊欄篩選選單。

## 🚀 如何使用

本專案採用純前端技術開發（Vanilla HTML/CSS/JavaScript），無須安裝任何後端環境或依賴套件。

1.  **下載/Clone 專案**：將專案檔案下載至本地端。
2.  **開啟檔案**：直接使用任何現代網頁瀏覽器（如 Chrome, Safari, Edge, Firefox）雙擊開啟專案中的 `index.html` 即可運行。

## 🛠 技術堆疊

*   **HTML5**: 語意化結構。
*   **CSS3**: 
    *   使用 CSS 變數 (Custom Properties) 統一管理設計系統與深淺色主題配色。
    *   純 CSS 實作平滑過渡動畫與 Grid/Flexbox 彈性排版。
*   **JavaScript (Vanilla)**: 處理 DOM 操作、資料過濾、分頁邏輯與狀態管理，無任何第三方框架依賴。

## 📂 資料結構說明

展商資料內建於 JavaScript 的 `DATA` 常數中，陣列的各欄位對應如下：
`[展商名稱, 類別, 優先度(Tier), 展位, 說明/切入點, 標籤(如"新"), 國家, 角色性質, Logo 圖片網址, 展商官方連結]`

如需更新資料，請直接使用文字編輯器開啟 HTML 檔案，尋找 `const DATA = [...]` 區塊並修改陣列內容即可自動套用至全站。
